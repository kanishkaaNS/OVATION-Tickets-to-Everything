# 🎟️ OVATION — Event Booking System

OVATION is a modern, responsive Event Ticket Booking System built entirely using **HTML5, CSS3, and Vanilla JavaScript**. It provides a complete frontend booking experience where users can browse events across major Indian cities, authenticate, select tickets, complete bookings, and generate printable receipts.

The project was developed to demonstrate how a feature-rich web application can be built using only client-side technologies while maintaining a clean architecture, responsive design, and an intuitive user experience.

## 🚀 Live Demo

🔗 https://ovation-tickets.netlify.app

## ✨ Features

### Event Discovery

* Browse events across 8 major Indian cities.
* Explore events by category.
* Persistent preferred city selection.
* Dynamic event rendering.

### Movie Booking

* Theatre selection.
* Showtime selection.
* Interactive ticket/seat selection.
* Live booking summary.

### Authentication

* User registration and login.
* Client-side validation.
* Persistent sessions using Local Storage.
* Protected checkout flow.

### Checkout

* Auto-filled user information.
* Credit card input formatting.
* Expiry date validation.
* Optional "Book for Someone Else" flow.
* Printable booking receipt.

### User Experience

* Responsive design.
* Smooth scrolling and animations.
* Modular UI components.
* Dynamic content rendering.
* Consistent design system.
